Objetivo:

Implementar y analizar modelos de aprendizaje no supervisado (K-means, DBSCAN, PCA y t-SNE) para segmentar  un problema creado por los estudiantes según el dataset donde están trabajadon.



Ejempl;o: segmentar perfiles de usuario o cliente en un entorno tecnológico. Visualizar resultados, comparar métodos y comunicar conclusiones de forma técnica y visual.

 Contexto del caso

Una plataforma digital de servicios personalizados desea segmentar a sus usuarios con base en datos de navegación, transacciones y comportamiento. El equipo de analítica necesita crear perfiles representativos usando métodos no supervisados para entender mejor el comportamiento y adaptar estrategias de marketing.
